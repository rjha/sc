Caveats

  • Ask for data before starting your tests
  • This is a live site so do not leave behind "ugly" data and posts like "test test" (cruel intentions)
  • Try to work with "right" test data that a "normal" user would also use.

Variables for testing

  • different browsers (IE9 vs. chrome)
  • sessions - Logged in user vs. anonymous user
  • Different login provider (3mik vs. Facebook)

Browsing

Browsing should support following cases

  • when you are not logged in
  • when you are logged in.
  • working in IE9 and IE9+
  • working in IE8
  • working in a sane browser like FF/Chromes

when you click on a tile, it shows you the details on an item details page

  • Text tile should show more link
  • Hover on an tile should show Like/Favorite buttons
