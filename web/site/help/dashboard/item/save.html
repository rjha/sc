How to save an item to view later?

  1. First, Go to an item details page.
  2. Item details page has a toolbar with icons just below the item title and above the item image
  3. click on "save"
  4. save this item to an existing list or create a new list for this item

To view all your lists, click on your name in main site toolbar and select Account. That will take you to the dashboard page. Please click on lists to view all your lists.
