How to add an item?

To create an item, click on the Add+ /Upload button. You will get a choice of

  1. Upload from computer
  2. Share a webpage
select option (1) if you want to upload images from your computer.
select option (2) if you want to share images from a webpage. This option will pull images from the webpage directly so you are saved the hassle of first downloading and then uploading the images.
