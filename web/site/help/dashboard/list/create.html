How to create a list?

To create a list from account page
  1. Sign in into your account
  2. click on lists link in menu
  3. Click on the + Create new list button.
  4. Type list name and description
  5. Click on Save
To create a list on item page
  1. click on Save inside the toolbar
  2. Clicking on save will open a popup
  3. You can select a list by clicking on it
  4. Or you can type a new list name and click create
