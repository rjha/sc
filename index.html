what happened to 3mik.com website?

We had fun running 3mik.com. However the fun can no longer justify the expenses that come with running a huge data driven site and a social network. We no longer have the bandwidth required to run 3mik.com. As of 31 August, 2013, we decided to pull the plug. However, the code is now in a public repository on Github. You can still have the fun to run a 3mik.com like site on your linux VM.

Can I use your code?

Absolutely. Please check the github repo for how to run the site on your own VM. We are releasing the code under Apache2 license. Have fun!
